# Research Report on Generative AI Applications

## Introduction
Generative AI (GenAI) is a branch of artificial intelligence that can create new things — like text, images, code, or even music.  
It’s developing really fast and is already used in many real-life situations.  
Below are several examples of where GenAI is applied and what benefits and challenges it brings.

---

## Use Case 1: Chatbots and Text Assistants
One of the most common uses of GenAI is in chatbots that can talk naturally with users.  
For example, ChatGPT helps people write texts, explain topics, or just have a conversation.

**Advantages:** saves time, reduces the workload of support teams.  
**Challenges:** sometimes gives inaccurate or made-up answers.  
**Examples:**  
- ChatGPT — https://chat.openai.com  
- Google Gemini — a smart assistant that understands text and images.

---

## Use Case 2: Image Generation
Models like DALL·E or Midjourney can create images based on a text prompt.  
You just describe what you want, and the AI generates a picture.

**Advantages:** very useful for designers and marketers; speeds up content creation.  
**Challenges:** issues with copyright, strange details, and limited control over the result.  
**Examples:**  
- DALL·E (https://openai.com/dall-e)  
- Midjourney (https://www.midjourney.com)

---

## Use Case 3: Code Generation
Developers use AI tools that can suggest or even write code automatically.  
For example, GitHub Copilot helps programmers by finishing code lines or generating functions.

**Advantages:** saves time, helps beginners learn faster.  
**Challenges:** AI code can contain bugs or security risks, so it always needs review.  
**Examples:**  
- GitHub Copilot  
- Amazon CodeWhisperer

---

## Use Case 4: Education
AI can support teachers and students by creating quizzes, lesson summaries, or personalized learning plans.  
It makes learning more interactive and flexible.

**Advantages:** saves time for teachers, makes studying more personal.  
**Challenges:** not all AI answers are correct, so human checking is still needed.  
**Examples:**  
- Khanmigo (Khan Academy)  
- ChatGPT for Education

---

## Conclusion
Generative AI is already a part of everyday life — from chatting and designing to coding and studying.  
It’s a powerful tool, but like any tool, it depends on how people use it.  
The key is to stay critical and use AI responsibly.
